Project Description

A tool to upload file directly into the GitHub Repository

Project Features

File Upload

"uploads file directly into the root of the repository of the GitHub account by using its API Token"

File Download

"Downloads a file directly from the root of the repository of the GitHub account"

Future Scope

"In the future, we aim to integrate this with an AI model to process the data from the file and obtain some usefull results."
